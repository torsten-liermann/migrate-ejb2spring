== Quick reference
This chapter summarizes common migrations as a brief orientation. The detailed explanations can be found in the thematic chapters.
[cols="2,2,3"]
|===
| Jakarta EE | Spring Boot | Note
| `@Stateless`
| `@Service`
| Check lifecycle and thread safety semantics
| `@Singleton`
| `@Service`
| EJB locks are no longer necessary and synchronization can be defined explicitly
| `@Stateful`
| Session/request-bound state models
| No direct replacement, technically remodeled
| `@TransactionAttribute`
| `@Transactional`
| Check propagation and rollback rules
| `@MessageDriven`
| `@JmsListener`
| Listener container, acknowledge mode, check concurrency
| `@Schedule`
| `@Scheduled`
| Check cron syntax and timing behavior
| `@Asynchronous`
| `@Async`
| Check proxy models and context propagation
| `@RolesAllowed`
| `@PreAuthorize`
| Check role prefixes and SecurityContext
| JAX-RS resources
| Spring MVC controllers
| Check path and exception mapping
| JSON-B
| Jackson
| Check annotations and serialization
| JNDI Lookup
| `@ConfigurationProperties`
| Transfer values ​​to `application.properties`
|===
