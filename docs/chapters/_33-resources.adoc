== Resources
This chapter collects materials that are helpful during migration. It does not replace any specification, but rather refers to the relevant sources and internal documents.
* Jakarta Enterprise Beans 4.0 specification
* Jakarta Persistence 3.x specification
* Spring Boot reference documentation
* Spring Framework Reference (Transactions, AOP, Bean Lifecycle)
* Migrating Enterprise Applications to Spring (SpringSource, 2008)
* Project-internal migration recipes and review reports
In this repository, additional analysis and planning documents are located in the `internal/` folder.
